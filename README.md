# README

*** a companion app for DrawThings: stash models & projects to an external disk to save space on your Mac system disk***

 - manage all DrawThings models types; image generation models, LoRAs, Controls, Embeddings
 - 'stash' models and projects to an external disk to save space on your main Mac system disk
 - Delete models 
 - Manage the display order for models; models, LoRAs, Controls
 - Create 'Stash Sets' - group of models and settings, eg 'WAN2.2', 'Flux Full' to stremaline DrawThings menus and save disk space
